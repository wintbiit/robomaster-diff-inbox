RoboMaster 2025机甲大师超级对抗赛赛季规划须知

2024-11-25

参与对象：通过了RMUC 2024规则测评的内地队伍、港澳台及海外队伍。甲级队伍及非甲级队伍均必须参与本环节

参与方式：

①内地队伍和港澳台队伍：以参赛队伍为单位，在 RoboMaster 论坛 ( https://bbs.robomaster.com/ ) 中完成撰写材料，通过队长账号登录 报名系统 ( https://registration.robomaster.com/registration-center ) 提交材料

②海外队伍：以参赛队伍为单位，队长填写问卷（后续更新）进行提交

提交时间：

①内地队伍：2024年12月9日18:00 - 2024年12月11日18:00（北京时间）

②港澳台及海外队伍：2024年12月11日18:00 - 2024年12月13日18:00（北京时间）

提交机会：1次

通过条件：获得“C”及以上的等级

考查内容：分为规划文档、进度安排、团队预算

请按公告附录中的赛季规划评审要求，在 R ( https://bbs.robomaster.com/ ) oboMaster 论坛 ( https://bbs.robomaster.com/ ) 战队功能中创建内容模板，选择对应的模板进行撰写，多配图表、流程图，数据充分，重点突出，逻辑清晰，文档命名：RMUC2025+校名+队名+版块名称

详细操作指引可参考赛季总结撰写指引： https://bbs.robomaster.com/article/16813

通过权限：

1、获得该环节对应的产品折扣，详情见《 RoboMaster 2025机甲大师超级对抗赛参赛手册 ( https://www.robomaster.com/zh-CN/resource/pages/announcement/1768 ) 》及《 RoboMaster 2025机甲大师高校系列赛物资购买说明 ( https://www.robomaster.com/zh-CN/resource/pages/announcement/1760 ) 》

2、获得裁判系统测评（基础测评）和中期进度考核的提交权限

3、开源赛季规划，且评分在所有开源战队中排名前2，可获得奖金5000元（税前）；排名3-10名，可获得奖金1000元（税前）；奖金将在赛季结束后统一处理。

4、由于赛季规划的内容与战队管理小组（含队长、副队长、项目管理）的工作内容高度相关，赛季规划的得分将作为年度优秀队长（团）及年度优秀项目管理评选的重要依据

最佳赛季规划奖申请流程：

1、根据开源规范开源赛季规划；

2、开源截止时间：12月23日18:00（北京时间）

3、填写申请问卷： https://djistore.wjx.cn/vm/h49XIGV.aspx

开源规范：

1、内地队伍和港澳台队伍：队长/领队老师登录 RoboMaster论坛 ( https://bbs.robomaster.com/portal.php ) ，在我的战队-赛季规划文档中将权限选择为公开或点击生成开源版本，修改后发布。帖子命名为：“【RMUC2025+校名+队名+赛季规划】”，帖子内可附开源文件；

2、海外队伍：将开源内容发送至官方邮箱robomaster@ dji.com ( http://dji.com/ ) ，邮件主题：“【RMUC2025+校名+队名+赛季规划 】”
